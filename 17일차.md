# 오늘의 수업


### stack(스택)
고정적인 메모리 구조로 먼저 들어간 데이터가 가장 마지막에 나오는 방식

### 함수 호출 규약

    cdecl
    
    - 함수 밖에서 스택을 정리
    - 사용자 함수
    
    stdcall
    
    - 함수 안에서 스택을 정리하고 리턴
    - win32 API
    
    fastcall
    
    - 함수 안에서 스택을 정리
    - 스택이 아닌 레지스터로 파라미터 전달
    - 64bit 시스템
